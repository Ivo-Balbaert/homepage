The Alan Programming Language

 

The alan compiler and runtime apply the possible parallelism in the users' code without the need for concurrent or asynchronous programming by enforcing predictable execution. alan has predictable execution by not allowing unbounded loops nor recursion. Learn more about alan through our blog, documentation, or source code.
